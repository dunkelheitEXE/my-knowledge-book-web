## What Is ORM?

It is a tool that helps to the developers interact with a database in the same way that they interact with [[OOP (Object-Oriented Programming)\|OOP (Object-Oriented Programming)]].

There are many Programming Languages and Frameworks where you will be able to find this kind of programming.
